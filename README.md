🌍 WanderEase Travel & Tours

✨ Project Description

WanderEase Travel & Tours is a static travel agency website built with HTML and CSS.
It showcases travel destinations, tour packages, and company info in a clean and modern layout.  

🛠 Features

🗂 Navigation Bar – Home, Destinations, Packages, About, Contact
🎯 Hero Section – Banner with tagline and “Book Now” button
🏝 Destination Cards – Grid layout with popular travel spots
💳 Tour Packages – Pricing cards with package details
💡 Why Choose Us – Highlights agency strengths (Affordable, Safe, Reliable)
📧 Contact Form – Simple form for inquiries
📱 Responsive Design – Works on all screen sizes


📌 Scope & Delimitations

Static website only (HTML & CSS)
“Book Now” button does not perform real bookings
No backend, database, or payment integration
Contact form does not store or send data

🎨 Wireframes

Layouts for homepage, packages, and details pages
Organized, user-friendly, and visually structured

🛠 Technologies

HTML5
CSS3 (Flexbox, Grid, Media Queries)
